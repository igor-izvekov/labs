Intel i9 11900KF

Краткое описание товара:

Intel Core i9-11900KF - 8 ядерный процессор компании Intel

Подробное описание товара:

Процессор Intel Core i9-11900KF стал обладателем 8 ядер и объемного трехуровневого кэша, который вместе с тактовой частотой 3.5 ГГц позволит системе быстро запускать различные программы и эффективно справляться с любыми задачами. Способность одновременно обрабатывать до 16 вычислительных потоков будет особенно актуальна для работы «тяжелого» софта. Созданный на базе архитектуры Intel Rocket Lake-S чипсет использует 14-нанометровый техпроцесс, который способствует оптимальной производительности и высокой энергоэффективности. Процессор Intel Core i9-11900KF предусматривает свободный множитель, который необходим для ручного разгона тактовой частоты и повышения вычислительной мощи системы. С его помощью можно вручную настроить частоту, при этом частота системной шины останется неизменной. В сборке с представленным чипсетом можно использовать оперативную память DDR4 объемом до 128 ГБ.

Характеристики:

  • Общее количество ядер: 8
  • Количество производительных ядер: 8
  • Количество энергоэффективных ядер: нет
